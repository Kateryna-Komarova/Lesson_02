Cascading Style Sheets - каскадные таблици стилей

Свойства обращенные через селектор по тегу коснутся всех элементов этой группы

Если мы хотим стилизовать конкретный элемент, то мы используем селектор по классу

CSS можно подключить 3 способами

  • Первый приоритетный - cоздать файл со стилями CSS
  • Второй - внутренний CSS чере тег style
  • Третий - инлайновый CSS. указываем стили внутри конкретного тега
